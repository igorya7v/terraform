# Provision AWS Network Layer for Staging and Production environments. 

Provision AWS Network Layer for Staging and Production environments using the Terraform Modules (see modules/aws_network/ folder).

## Components
### Staging
  * VPC
  * Internet Gateway
  * Public Subnets
  * NAT Gateways
